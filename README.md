# Frontend Task

A clean and responsive landing page built using **HTML**, **CSS**, and **JavaScript**.  
This project was created as part of a frontend assignment to practice layout design, user interface structure, and basic interactivity.

---

##  Features

- Fully responsive layout  
- Modern and clean UI  
- Smooth CSS animations and transitions  
- Section-based design (Hero, About, Services, Contact, etc.)  
- Interactive elements powered by JavaScript  
- Cross-browser compatibility  

---

##  Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Interactivity and DOM handling
